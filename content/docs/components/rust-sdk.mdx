Rust SDK 用于在不依赖数据面代理的情况下接入服务治理能力，适合需要 Proxyless Service Governance 的 Rust 应用。

## 安装

```toml
[dependencies]
pole_rust = "0.2.0"
```

## 环境要求

- Rust 版本要求：`>= 1.63.0`。
- 现有 README 中建议通过 `cargo bench` 自行运行 benchmark。
