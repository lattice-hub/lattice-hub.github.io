Specification 仓库定义服务治理标准和跨语言协议。它遵循下一代架构基金会的服务治理标准，并在此基础上扩展 AI Native 相关能力。

## 标准范围

- 服务管理：namespace、service、instance、health check。
- 流量治理：router、load balancer、limiter。
- 容错：circuit breaker、fault detector、retry。
- 访问控制：authentication、authorization。
- AI Native：MCP server proto。

## 跨语言构建

现有 README 记录了 Java、Go、Rust 三种语言的构建入口：

- Java：`source/java/pole-specification`
- Go：`source/go`
- Rust：`source/rust/pole-specification`

## 版本注意

现有文档中 Java/Go 依赖示例写 `v1.3.0`，Rust 总 README 写 `pole-specification = "1.3"`，Rust 子 README 写 `0.1.0`。发布前建议统一版本表述。
