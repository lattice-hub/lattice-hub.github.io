Console 是 Lattice Hub 的管理入口。它为服务治理资源提供可视化操作界面，降低服务、配置、命名空间、用户与权限策略的维护成本。

## 当前定位

- 管理服务、命名空间和配置文件。
- 支持用户与权限管理。
- 为控制面能力提供统一的可视化入口。

## 技术栈

现有前端工程基于 TDesign React Starter，使用 React 与 Vite，包含中后台常见页面结构、暗黑模式、主题色、空间布局和 Mock 数据方案。

## 首版官网呈现方式

首页中 Console 作为生态矩阵的一部分出现。详细交互截图和操作手册后续可以从 `../pole-console` 的实际页面继续补充。
